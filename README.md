# Hand Gesture Servo & LED Control

Real-time computer vision system that controls a servo angle and LED state using hand gestures detected via webcam


##  Features
- Hand tracking with MediaPipe
- Servo angle control
- LED control via hand gestures
- Serial communication with Arduino
- Real-time feedback

##  Structure
arduino/ → Arduino firmware  
python/ → Computer vision control  
docs/ → circuit photos and diagrams  
media/ → thumbnails and assets
